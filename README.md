# Library_PROJECT_4.0

Library_Project is a simple desktop application in C#, that allows us to manipulate data.



Application is based on:

* CRUD (Create, Read, Update, Delete) functionality,
* SQL LocalDB,
* Entity Framework for C#,
* DAL (Data Access Layer).
